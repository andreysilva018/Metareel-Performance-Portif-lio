# 🚀 Metareel Performance

Landing page institucional desenvolvida para apresentar a proposta da Metareel Performance, uma agência focada em estratégias de vendas, marketing digital e uso de CRM para crescimento de farmácias.

---

## 🧠 Sobre o projeto

O objetivo deste projeto é criar uma presença digital profissional para a empresa, destacando:

- Problemas comuns enfrentados por farmácias
- Soluções baseadas em marketing e tecnologia
- Uso de CRM para aumentar a recorrência de clientes
- Estratégias focadas em crescimento de faturamento

---

## 🎯 Objetivo

Demonstrar de forma clara e direta como a Metareel Performance ajuda farmácias a:

- Aumentar o número de clientes
- Melhorar a retenção
- Gerar vendas recorrentes
- Organizar o processo de atendimento e vendas

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
